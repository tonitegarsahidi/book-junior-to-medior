# 0 - FROM JUNIOR TO MEDIOR -  a Mukaddimah Series

Setiap lihat tentang impression tulisan saya, saya mendapati bahwa sekitar 30-40% audiens dan follower saya adalah temen-temen IT yang berada di Entry Level. Entry level disini tentu secara jam terbang saja ya, bukan skill. Karena soal skill saya yakin banyak dari mereka yang jauh lebih jago daripada saya. Anak zaman now tuh pada ngerih-ngerih skill-nya. Saya cuman -kebetulan- lebih lahir duluan aja. 

Jadi biar LinkedIn saya pun “tetap” terlihat sebagai LinkedIn anak IT beneran (yeah saya masih anak-anak), maka saya izin sesekali untuk posting hal-hal yang tetap berbau teknikal. Semoga kakak-kakak senior tak bosan, dan temen-temen non IT, tidak berkeberatan.

Yang saya share pun bukan hal yang baru, beberapa dari kalian mungkin sudah dapat di kelas, bootcamp, ataupun dari dunia lain. Tapi mungkin juga belum. Ketika 4 tahun lalu saya memutuskan masuk kembali ke dunia profesional IT di korporasi, tentu ada banyak hal “baru tapi lama” yang saya dapati. Ada yang sebelumnya belum pernah saya dapatkan, ataupun dapat tapi belum pernah saya praktikkan di karir sebelumnya di dunia freelance dan juga akademisi.
